class LuaJIT

package hxluajit

@:directlyUsed@:buildXml("<include name=\"${haxelib:hxluajit}/project/Build.xml\" />")@:include("lua.hpp")@:unreflective

Provides access to various properties and functionalities of LuaJIT.

Static variables

@:native("::String(LUAJIT_COPYRIGHT)")staticCOPYRIGHT:String

The copyright information of LuaJIT.

@:native("LUAJIT_MODE_ALLFUNC")staticMODE_ALLFUNC:Int

Recurse into subroutine protos.

@:native("LUAJIT_MODE_ALLSUBFUNC")staticMODE_ALLSUBFUNC:Int

Change only the subroutines.

@:native("LUAJIT_MODE_DEBUG")staticMODE_DEBUG:Int

Set debug mode (idx = level).

@:native("LUAJIT_MODE_ENGINE")staticMODE_ENGINE:Int

Set mode for the whole LuaJIT engine.

@:native("LUAJIT_MODE_FLUSH")staticMODE_FLUSH:Int

Flush JIT-compiled code.

@:native("LUAJIT_MODE_FUNC")staticMODE_FUNC:Int

Change mode for a function.

@:native("LUAJIT_MODE_MASK")staticMODE_MASK:Int

Bitmask for LuaJIT mode settings.

@:native("LUAJIT_MODE_MAX")staticMODE_MAX:Int

Maximum mode value.

@:native("LUAJIT_MODE_OFF")staticMODE_OFF:Int

Turn feature off.

@:native("LUAJIT_MODE_ON")staticMODE_ON:Int

Turn feature on.

@:native("LUAJIT_MODE_TRACE")staticMODE_TRACE:Int

Flush a compiled trace.

@:native("LUAJIT_MODE_WRAPCFUNC")staticMODE_WRAPCFUNC:Int

Set wrapper mode for C function calls.

@:native("::String(LUAJIT_URL)")staticURL:String

The URL pointing to LuaJIT's homepage.

@:native("::String(LUAJIT_VERSION)")staticVERSION:String

The version string of LuaJIT.

@:native("LUA_VERSION_NUM")staticVERSION_NUM:Int

The version number of LuaJIT.

Static methods

@:native("LUAJIT_VERSION_SYM")staticVERSION_SYM():Void

Enforces dynamic linker errors for LuaJIT version mismatches. Call from main.

@:native("luaJIT_profile_dumpstack")staticprofile_dumpstack(L:RawPointer<Lua_State>, fmt:ConstCharStar, depth:Int, len:Star<SizeT>):ConstCharStar

Dumps the LuaJIT execution stack for profiling.

Parameters:

L

The Lua state.

fmt

The format string.

depth

The maximum depth of the stack to dump.

len

Pointer to store the length of the dump.

Returns:

A const char pointer to the dumped stack, or null on error.

@:native("luaJIT_profile_start")staticprofile_start(L:RawPointer<Lua_State>, mode:ConstCharStar, cb:LuaJIT_Profile_Callback, data:RawPointer<Void>):Void

Starts profiling LuaJIT execution.

Parameters:

L

The Lua state.

mode

The profiling mode.

cb

The callback function for profiling.

data

Additional data to pass to the callback.

@:native("luaJIT_profile_stop")staticprofile_stop(L:RawPointer<Lua_State>):Void

Stops profiling LuaJIT execution.

Parameters:

L

The Lua state.

@:native("luaJIT_setmode")staticsetmode(L:RawPointer<Lua_State>, idx:Int, mode:Int):Int

Controls the LuaJIT engine mode.

Parameters:

L

The Lua state.

idx

The stack index or 0.

mode

The mode to set.

Returns:

1 on success, or 0 on failure.
